# Gate: 优化

网站已经跑起来了。想让它更好？选一个方向。

## 当前阶段

| 阶段 | 目标 | 对应的技能 |
|------|------|-----------|
| **Level 1** ✅ | 能跑、有 SEO、已上线 | （已完成） |
| **Level 2** | 快、稳、全覆盖 | 性能 · 无障碍 · 测试 · CI |
| **Level 3** | 精致、自动化、有数据 | 动效 · 监控 · 分析 |

---

## Level 2 — 让网站更专业

> `/gate-perf` — ⚡ 性能优化
> 
> 加载 `performance` · 提升 Lighthouse 分数 · Core Web Vitals

> `/gate-a11y` — ♿ 无障碍
> 
> 加载 `accessibility` · WCAG AA 标准 · 键盘导航 · 屏幕阅读器

> `/gate-testcov` — 🧪 测试覆盖
> 
> 加载 `e2e-playwright` · `ai-testing` · 加 E2E 测试 · 组件测试

> `/gate-ci` — 🔄 CI/CD
> 
> 加载 `vercel-deploy` · GitHub Actions · 自动部署 · 预览部署

## Level 3 — 让网站更出色

> `/gate-animate` — ✨ 动效与交互
> 
> 加载设计 skills · Framer Motion · 滚动动画 · 过渡效果

> `/gate-monitor` — 📊 监控与分析
> 
> Vercel Analytics · Error tracking · 用户行为 · 性能监控
