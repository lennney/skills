name: seo-checklist
description: SEO 完整检查。网站上线前后加载。你在后台逐项检查，只问用户不知道的内容（关键词、网站描述）。
---

用户不需要懂 SEO。问 2 个问题：
1. "用哪几个关键词？" — 用户说 3-5 个词
2. "网站一句话介绍？" — 用作 description

然后在后台逐项检查并修复：

**技术项（自动做）：**
- 每页 `<title>` 含关键词，50-60 字符
- 每页 `meta description` 120-155 字符
- 每页一个 `<h1>`，层级不乱
- 图片有 alt + 语义化文件名 + lazy loading
- Sitemap + Robots 正确
- canonical URL 正确

**建议项（告诉用户）：**
- "建议每页正文至少 300 字，当前首页只有 X 字"
- "建议提交 Google Search Console"
- "建议用 opengraph.xyz 预览分享效果"

做完告诉用户 SEO 状态："全部没问题"或"有几个建议项"。不需要展示技术细节。
