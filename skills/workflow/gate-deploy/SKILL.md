---
name: gate-deploy
description: 部署阶段。SEO 配好后加载。帮用户上线，用大白话解释每步在做什么。
---

告诉用户："准备上线了，我来帮你搞定。"

1. **构建验证** → `npm run build`
   "先跑一遍构建，确保所有代码没问题。"

2. **推 GitHub**
   "把你网站的代码推到 GitHub 上存起来——相当于一个保险箱，代码不会丢，以后想改什么也方便。"

3. **Vercel 部署**（引导用户在 Vercel 导入项目）
   "Vercel 是专门托管 Next.js 网站的服务，你点一下 Import，它就会自动从 GitHub 拉代码、部署到服务器上。"

4. **域名**（如需自定义域名问用户）
   "你有自己的域名吗？有的话告诉我，我告诉你怎么配。"

部署完成后说："网站上线了！地址是 X。接下来可以让它变得更好——要不要做性能优化？" → `/gate-optimize`
