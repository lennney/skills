# Anti-Patterns: What Makes Sites Look AI-Generated

## Typography Sins

| Offender | Why It's Bad |
|----------|-------------|
| **Inter / Roboto** | The default AI font. Used in ~80% of AI-generated UIs. |
| **Space Grotesk** | The "I want to look modern but have no taste" choice. |
| **System-ui stack** | Lazy. Shows no intentional typography choice. |
| **Bounce easing** | `cubic-bezier(0.68, -0.55, 0.265, 1.55)` — the AI signature ease. |
| **Monospace for "developer vibes"** | An overused shorthand for "tech." |

## Color Sins

| Offender | Why It's Bad |
|----------|-------------|
| **Purple-to-blue gradient** | The #1 AI color scheme. Used by every AI tool. |
| **Purple-on-white** | The default "modern SaaS" AI look. |
| **Pure black (#000) backgrounds** | Never looks premium. Always use off-black. |
| **Gray-on-gray text** | Low contrast, unreadable, lazy. |
| **Too many colors** | AI tends to use 4-5 evenly distributed colors. Pick 2-3 max. |

## Layout Sins

| Offender | Why It's Bad |
|----------|-------------|
| **Cards inside cards** | The "nesting doll" layout. Every AI site does this. |
| **Every section: centered heading + grid below** | Template-like. Vary your section structures. |
| **Identical spacing between all sections** | No rhythm. Use varied spacing (tight, spacious, tighter). |
| **Symmetry everywhere** | Perfect symmetry reads as template. Asymmetry reads as designed. |

## Component Sins

| Offender | Why It's Bad |
|----------|-------------|
| **Emoji as icons** | ★ 👍 👎 ☰ — cheap. Use SVGs. |
| **Gradient text headings in every section** | One gradient heading is a statement. Five is a template. |
| **Standard rounded buttons only** | Mix sharp, pill, bordered, underlined CTA shapes. |
| **Avatar-style testimonials** | The "photo + name + quote" card is the #1 AI testimonial pattern. |
| **Card shadows on everything** | Creates visual noise. Use sparingly. |
